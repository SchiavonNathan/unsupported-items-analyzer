# ItensNotSuported-Analyser

Projeto simples para analisar itens não suportados e gerar um relatório.

## Descrição

Este repositório contém um script que processa dados e gera um relatório (veja `report.csv`). Há também um template HTML em `templates/index.html` usado para visualização.

## Pré-requisitos

- Python 3.8+
- Dependências listadas em `requirements.txt`

## Instalação

Instale dependências:

```bash
pip install -r requirements.txt
```

## Uso

Execute o script principal:

```bash
python main.py
```

Edite `report.csv` ou os arquivos de entrada conforme necessário antes de rodar.

## Estrutura

- `main.py` — ponto de entrada
- `report.csv` — arquivo de dados / exemplo de saída
- `templates/index.html` — template para relatório
